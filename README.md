### App Features

1. Match schedules based on choosen league.

 * [x] List of upcoming matches.
 * [x] List of last matches.
 * [x] Favorite match.
 * [x] Match search.
 * [x] Match Detail.
 * [x] Adding reminder for upcoming matches to your calendar.
